
# Distributed Job Scheduler - Codity Assignment

## Overview
This repository implements a **production-grade distributed job scheduling platform** as described in the academic assignment. It includes:
- **Backend** вЂ“ FastAPI (Python) with JWT authentication, PostgreSQL, and a separate worker service.
- **Frontend** вЂ“ React + Vite (TypeScript) dashboard with a modern, premium UI.

- **Docker Compose** вЂ“ Spins up PostgreSQL, the FastAPI API, worker service, and the React UI.

## Features
- User authentication (JWT) and authorization
- Organization and project management
- Queue management (priority, concurrency limits, pause/resume)
- Job creation (immediate, delayed, scheduled)
- Worker service for job execution
- Job lifecycle tracking (queued в†’ claimed в†’ running в†’ completed/failed)
- Retry policies
- Dead Letter Queue (DLQ) support
- Dashboard with job statistics and visualizations

## Quick Start
### Using Docker Compose (Recommended)
```powershell
# From the repository root
docker-compose up --build
```

### Local Development
#### Backend
```powershell
# Create a virtual environment
python -m venv venv
.\venv\Scripts\activate

# Install dependencies
pip install -r backend/requirements.txt

# Start the backend server
uvicorn backend.src.app.main:app --reload
```

#### Frontend
```powershell
cd frontend
npm install
npm run dev
```
